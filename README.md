# 🎨 Color Switcher

An interactive background color switcher built with **React 19**, **Vite**, and **Tailwind CSS v4**.

## 🚀 Getting Started

```bash
git clone https://github.com/heetnandu01/color-Switcher.git
cd color-Switcher
npm install
npm run dev
```
